# InsaneOne.Core.Injection

Lightweight dependency injection / service locator for Unity, extracted from [InsaneOne Core](https://github.com/OlegDzhuraev/Core).

## Install

Via Unity Package Manager → Add package from git URL:

```
https://github.com/OlegDzhuraev/InsaneOne.Core.Injection.git
```
